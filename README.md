# Praticando C 🎯

Um repositório de exercícios práticos de programação em linguagem C, cobrindo conceitos fundamentais e intermediários.

---

## 📚 Conteúdo do Repositório

### Conceitos Básicos
- **basicao.c** - Entrada, processamento e saída básica
- **helloworld15.c** - Primeiro programa em C
- **helloworldarq.c** - Escrita em arquivo

### Operações Matemáticas
- **circuferenc.c** - Cálculo de área de círculo
- **cestodmacas.c** - Cálculo de percentual
- **permut.c** - Cálculo de permutação/fatorial
- **somatorio.c** - Cálculo de somatório
- **drink.c** - Cálculo de percentuais de ingredientes

### Estruturas de Controle
- **decrescif.c** - Ordenação decrescente de três números
- **maiorf.c** - Encontrar maior entre dois números
- **maiortrs.c** - Encontrar maior entre três números
- **maiordpzero.c** - Encontrar maior número digitado
- **divisvtrslst.c** - Verificar divisibilidade
- **divsp3.c** - Verificar divisibilidade por 3
- **multiplo0.c** - Encontrar múltiplos
- **precoprdesc.c** - Cálculo de desconto
- **semaforo.c** - Switch case com validação
- **semaforO.cpp** - Variação com loop infinito

### Arrays e Vetores
- **cncposnum.c** - Preenchimento e exibição de vetor
- **umdstrsqtr.c** - Vetor com 4 elementos
- **buscasequencial.c** - Busca sequencial em vetor
- **vtrndnrptd.c** - Vetor com números não repetidos
- **vtrrndrptd.c** - Vetor aleatório com arquivo

### Matrizes
- **mtrzquadord3.c** - Matriz quadrada de ordem 3
- **metamatriz.c** - Exibição de matriz
- **mtrzestrnhboa.c** - Entrada de matriz
- **mtrztranspstasimp.c** - Transposição simples
- **mtrztranspstaext.c** - Transposição estendida
- **matrizcub.c** - Matriz tridimensional

### Algoritmos Avançados
- **fibonnacci.c** - Série de Fibonacci
- **mediaf.c** - Cálculo de média com situação
- **umdstrsqtrdc.c** - Ordenação por insertion sort
- **nascsenha.c** - Processamento de data de nascimento
- **loooloucao.c** - Loop simples com incremento

### Outros
- **vdsff.c** - Arquivo vazio/em desenvolvimento

## 🚀 Como Usar

1. **Compilar um arquivo:**
   ```bash
   gcc nome_arquivo.c -o nome_arquivo
   ```

2. **Executar o programa:**
   ```bash
   ./nome_arquivo
   ```

3. **Compilar com bibliotecas específicas:**
   ```bash
   gcc nome_arquivo.c -o nome_arquivo -lm  # Para funções matemáticas
   ```
